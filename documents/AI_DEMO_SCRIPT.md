# Aura AI 系统演示脚本

## 🎯 演示目标
展示 Aura 智能系统的 10 大核心 AI 能力，包括多 Agent 协作、RAG 检索、Function Calling 等先进技术。

## 📋 演示场景列表

### 1. 产品推荐能力
**用户输入：** "我需要一个智能香薰机，适合卧室使用"
**系统响应：** 分析用户需求，调用 ProductExpertAgent，推荐适合卧室的香薰机产品
**技术亮点：** 意图识别 + 产品专家 Agent

### 2. 产品说明书查询
**用户输入：** "Aura S1 香薰机的使用方法"
**系统响应：** 调用 QueryProductManualFunction，从产品说明书中检索相关信息
**技术亮点：** RAG 检索 + PDF 向量化

### 3. 订单状态查询
**用户输入：** "我的订单 AURA20240001 的状态是什么？"
**系统响应：** 调用 GetOrderStatusFunction，查询并返回订单状态
**技术亮点：** Function Calling + 订单服务集成

### 4. 订单地址修改
**用户输入：** "我想修改订单 AURA20240001 的收货地址"
**系统响应：** 调用 UpdateOrderAddressFunction，更新订单地址
**技术亮点：** 多轮对话 + 函数调用

### 5. 产品库存查询
**用户输入：** "Aura S1 香薰机还有库存吗？"
**系统响应：** 调用 CheckInventoryFunction，查询产品库存状态
**技术亮点：** 实时数据查询 + 函数调用

### 6. 产品搜索能力
**用户输入：** "搜索所有带有睡眠模式的产品"
**系统响应：** 调用 SearchProductsFunction，返回符合条件的产品列表
**技术亮点：** 语义搜索 + 产品服务集成

### 7. 多 Agent 协作
**用户输入：** "我想了解 Aura S1 的功能，并且查询我的订单状态"
**系统响应：** OrchestratorAgent 协调 ProductExpertAgent 和 CustomerServiceAgent
**技术亮点：** 多 Agent 协调 + 任务分解

### 8. 上下文理解能力
**用户输入：** "它有什么颜色可选？"
**系统响应：** 基于之前的对话上下文，理解用户询问的是 Aura S1 的颜色选项
**技术亮点：** 上下文记忆 + 意图理解

### 9. 自然语言对话
**用户输入：** "你好，很高兴认识你"
**系统响应：** 友好回应，保持品牌语调
**技术亮点：** 自然语言处理 + 品牌一致性

### 10. 复杂问题解答
**用户输入：** "比较 Aura S1 和 Aura X2 的区别"
**系统响应：** 调用 ProductExpertAgent，提供详细的产品比较
**技术亮点：** 产品专家知识 + 客观比较

## 🛠️ 技术架构展示

### 核心组件
1. **OrchestratorAgent**：主协调者，负责意图识别和 Agent 分配
2. **ProductExpertAgent**：产品专家，提供产品信息和推荐
3. **CustomerServiceAgent**：客服专家，处理订单和售后
4. **RAGService**：文档检索服务，基于向量存储
5. **MultiAgentService**：多 Agent 协调服务
6. **MemoryService**：三层记忆系统，保存对话上下文

### 技术栈
- Spring Boot 3.2
- Spring AI 1.0
- OpenAI API
- JPA + MySQL
- Apache PDFBox

## 🎨 品牌语调
全程保持 Aura 的品牌语调：温暖、有机、精致、平静，避免过于技术化的表达。
