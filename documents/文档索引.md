## 🎯 快速导航

### 新成员入职
1. 阅读 [README_CN.md](./README_CN.md) 了解项目
2. 阅读 [团队分工详细方案.md](./团队分工详细方案.md) 了解任务
3. 阅读 [团队分工与Git工作流.md](./团队分工与Git工作流.md) 了解协作流程
4. 阅读 [开发规范.md](./开发规范.md) 了解代码规范

### 开始开发
1. 查看 [团队分工详细方案.md](./团队分工详细方案.md) 确认负责的类
2. 遵循 [团队分工与Git工作流.md](./团队分工与Git工作流.md) 的 Git 流程
3. 参考 [ARCHITECTURE.md](./ARCHITECTURE.md) 了解架构设计
4. 遵循 [开发规范.md](./开发规范.md) 编写代码

### 遇到问题
1. 查看 [README_CN.md](./README_CN.md) 的故障排除部分
2. 查看 [ARCHITECTURE.md](./ARCHITECTURE.md) 的相关章节
3. 在团队群里提问
